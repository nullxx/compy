NAME

genl − generic netlink utility frontend

SYNOPSIS

genl [options]{ctrl}[help]

DESCRIPTION

This manual page documents briefly the genl command.

genl utility provides a simple frontend to the generic netlink library.

OPTIONS

genl supports the following options.
−h, −help

Show summary of options.

−V, −Version

Show version of program.

−s, −stats, −statistics

Show object statistics.

−d, −details

Show object details.

−r, −raw

Dump raw output only.
